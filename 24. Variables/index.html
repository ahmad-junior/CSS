CSS Variables

Variables are a way to store information to be reused throughout a document. They are declared by using the var() function, followed by the variable name inside of a pair of parentheses.

Variables are case-sensitive, so --main-color and --Main-Color are two different variables.

Variables can be declared in the root of the document, or in a specific selector. If a variable is declared in a selector, it will only be available to that selector and its children.

Variables can be used in any CSS property value, not just custom properties.

var()

Variables are declared by using the var() function, followed by the variable name inside of a pair of parentheses.

Syntax:
    Declaration:
        --variable-name: value;
    Accessing:
        var(--variable-name);

Overiding Variables

Variables can be overriden by using the !important keyword.

Syntax:
   var(--variable-name, !important)

We can overwrite the global and local variables

Variables and JavaScript

Variables can be used in JavaScript to change the value of a variable.

Syntax:
    const root = document.documentElement;
    root.style.setProperty('--variable-name', 'value');

Queries

Queries are a way to change the value of a variable based on a certain condition.

Syntax:
    var(--variable-name, condition)
