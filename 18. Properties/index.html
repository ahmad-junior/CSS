CSS Counters

CSS counters are "variables" maintained by CSS whose values can be incremented by CSS rules (to track how many times they are used). Counters let you adjust the appearance of content based on its placement in the document.

To work with css counters we have the following css Properties:
  • counter-reset - Creates or resets a counter
  • counter-increment - Increments a counter value
  • content - Inserts generated content
  • counter() or counters() function - Adds the value of a counter to an element

Units

Units are used to specify the size of a CSS property. There are many different units available in CSS, but the most common are:

  • px - Pixels
  • em - Relative to the font-size of the element (2em means 2 times the size of the current font)
  • rem - Relative to font-size of the root element
  • % - Relative to the parent element
  • vh - Relative to 1% of the viewport's height
  • vw - Relative to 1% of the viewport's width
  • vmin - Relative to 1% of viewport's smaller dimension
  • vmax - Relative to 1% of viewport's larger dimension

!important

The !important rule is used to override any other declarations. It should be used with caution, because it is often considered bad practice. It is better to be specific when selecting elements than using !important.

Example:

        h1 {
            color: red !important;
        }

Math functions

Math functions are used to perform calculations when declaring CSS values. The following math functions are available:

  • calc()
  • min()
  • max()
  • clamp()

Example:

        h1 {
            width: calc(100% - 50px);
        }
