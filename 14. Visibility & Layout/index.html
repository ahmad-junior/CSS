Visibility and Layouts

Display

The display property is the most important CSS property for controlling layout. It can turn an inline element into a block-level element, or vice versa, and it can hide an element completely.

The display property accepts several values, including block, inline, inline-block, none, and list-item. The display property controls how an element is displayed on the page. The default value of the display property is inline, which means that the element is displayed as an inline element. The element will not start on a new line and will not affect the layout of the elements that come after it.

Blok-Level Elements

A block-level element always starts on a new line and takes up the whole width available (stretches out to the left and right as far as it can). Examples of block-level elements are:

  • div
  • p
  • ul
  • li
  • form
  • nav
  • header
  • footer
  • section
  • h1 ... h6

Inline Elements

An inline element does not start on a new line and only takes up as much width as necessary. Examples of inline elements are:

  • span
  • a
  • img
  • input
  • button
  • label
  • select
  • textarea

Display None

The display property can also be set to none, which hides the element. This is useful when you want to hide an element without removing it from the DOM. For example, you can hide an element until it is needed, or you can hide an element until it has been styled.

Position

The position property specifies the type of positioning method used for an element.

  1. static
  2. relative
  3. fixed
  4. absolute
  5. sticky

z-index

The z-index property specifies the stack order of an element. An element with greater stack order is always in front of an element with a lower stack order.

Overflow

The overflow property specifies what should happen if content overflows an element's box.

  1. visible
  2. hidden
  3. scroll
  4. auto

Inline-block

The inline-block value displays an element as an inline-level block container. The element itself is formatted as an inline element, but you can apply height and width values.

Align

The align property specifies the horizontal alignment of an element.

  1. left
  2. right
  3. center
  4. justify

Opacity

The opacity property specifies the opacity/transparency of an element.

Website Layout

There are many different ways to create a website layout. The most common way is to use a combination of HTML tables and CSS.

Float

The float property is used for positioning and formatting content e.g. let an image float left to the text in a container.

Clear

The clear property specifies what elements can float beside the cleared element and on which side.

Flexbox

The flexbox layout is more efficient than the table layout, and it offers more flexibility. The flexbox layout is most appropriate to the components of an application, and small-scale layouts, while the grid layout is intended for larger scale layouts.
