Outline

An outline is a line drawn outside the element's border.

It's like margin.

Outline outbox

Properties

Property Description
outline Shorthand property for all the outline properties
outline-color Specifies the color of an outline
outline-style Specifies the style of an outline
outline-width Specifies the width of an outline

Values

Value Description
auto Browser calculates the width of the outline
thin Thin outline
medium Medium outline
thick Thick outline
length Specifies the width of the outline in px, pt, cm, etc.
color Specifies the color of the outline
none No outline
dotted Dotted outline
dashed Dashed outline
solid Solid outline
double Double outline
groove 3D grooved outline
ridge 3D ridged outline
inset 3D inset outline
outset 3D outset outline
