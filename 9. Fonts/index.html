CSS Fonts

Selection of Good fonts is important for a website. The font you choose for your website should be easy to read, and should match the overall feel of the site.

Font Families

Font families are a group of fonts that are similar in style. The browser will display the first font in the list if it is available on the user's computer. If the first font is not available, it will try the next font, and so on.

Font Size

The font-size property specifies the size of the font. The font-size property can have the following values:

  • xx-small
  • x-small
  • small
  • medium
  • large
  • x-large
  • xx-large
  • smaller
  • larger
  • length
  • percentage

Font Weight

The font-weight property specifies the weight of a font. The font-weight property can have the following values:

  • normal
  • bold
  • bolder
  • lighter
  • 100
  • 200
  • 300
  • 400
  • 500
  • 600
  • 700
  • 800
  • 900

Font Style

The font-style property specifies the font style for a text. The font-style property can have the following values:

  • normal
  • italic
  • oblique

Font Variant

The font-variant property specifies whether or not a text should be displayed in a small-caps font. The font-variant property can have the following values:

  • normal
  • small-caps

Text Transform

The text-transform property specifies how to capitalize an element's text. The text-transform property can have the following values:

  • capitalize
  • uppercase
  • lowercase
  • none

Text Decoration

The text-decoration property specifies the decoration added to text. The text-decoration property can have the following values:

  • none
  • underline
  • overline
  • line-through
  • blink

Syntax

  • font-family: value;
  • font-weight: value;
  • font-style: value;
  • font-variant: value;
  • font-size: value;
  • line-height: value;
  • font: font-style font-variant font-weight font-size/line-height font-family;
