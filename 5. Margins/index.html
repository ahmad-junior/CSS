Margins

Margins are used to create space around elements, outside of any defined borders.

Margins can have different values for each side of an element (top, right, bottom, and left).

If you only specify one value, it will be applied to all four sides.

If you specify two values, the top and bottom borders will be set to the first value, and the right and left will be set to the second.

If you specify three values, the top will be set to the first value, the left and right will be set to the second, and the bottom will be set to the third.

Margin Properties

The margin properties are used to create space around elements, outside of any defined borders.

There are four margin properties: margin-top, margin-right, margin-bottom, and margin-left.

The margin properties can have the following values:

  • auto - The browser calculates the margin
  • length - Defines a margin in px, pt, cm, etc. Negative values are allowed
  • % - Defines a margin in % of the width of the containing element
  • inherit - Inherits the margin value from the parent element

Margin Shorthand

The margin shorthand property sets all four margin properties in one declaration.

The margin shorthand property can have from one to four values:

  • margin: 25px 50px 75px 100px;
  • margin: 25px 50px 75px;
  • margin: 25px 50px;
  • margin: 25px;
