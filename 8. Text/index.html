Text

Color & Background

Color and Background is used to change the color of the text and background of the text.

Text Align

Text Align is used to align the text to the left, right, center or justify.

Text Decoration

Text Decoration is used to add decoration to the text like underline, overline, line-through, blink.

Text Transform

Text Transform is used to change the case of the text like uppercase, lowercase, capitalize.

Text Shadow

Text Shadow is used to add shadow to the text.

Text Indent

Text Indent is used to indent the text.

Text Overflow

Text Overflow is used to show the text that is overflowed.

Text Wrap

Text Wrap is used to wrap the text.

Text Spacing

Text Spacing is used to add spacing between the letters, words and lines.

Text Direction

Text Direction is used to change the direction of the text.

Text Alignment

Text Alignment is used to align the text.

Text Justify

Text Justify is used to justify the text.

Text Line Height

Text Line Height is used to set the line height of the text.
