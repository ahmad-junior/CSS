Float

The CSS float property specifies how an element should float.

Float is a CSS positioning property. An element can be floated to the left or right of its container, allowing text and inline elements to wrap around it.

Float is often used to create entire web layouts.

We use following float properties to design our pages

  • float: left;
  • float: right;
  • float: none;
  • float: inherit;

Clear

Clear is a CSS positioning property. An element can be cleared to the left or right of its container, allowing text and inline elements to wrap around it.
