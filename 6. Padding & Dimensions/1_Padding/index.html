Difference Between Margin and Padding

Margin

  • Margin is used to create space outside the element.
  • Margin does not affect the background color or border.
  • Margin does not affect the width and height of an element.

Padding

  • Padding is used to create space inside the element.
  • Padding affects the background color or border.
  • Padding affects the width and height of an element.
